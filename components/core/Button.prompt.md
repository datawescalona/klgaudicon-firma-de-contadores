Confident geometric pill button with a soft "sticker" lift on hover — the core action element for KLG AUDICON.

```jsx
<Button variant="cta" size="lg" icon={<WhatsAppIcon />}>Conciliación en 24h</Button>
```

Variants: `primary` (cobalt, default), `cta` (bronze — reserve for conversion moments like contact/WhatsApp), `secondary` (cobalt outline), `ghost`. Sizes `sm | md | lg`. Pass `icon` + `iconRight` for icon placement, `fullWidth` for mobile bars, `href`/`as="a"` to render a link.
